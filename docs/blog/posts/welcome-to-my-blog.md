## Why Material for MkDocs?

I chose Material for MkDocs for several reasons:

- **Beautiful Design**: Modern, clean interface that looks great out of the box
- **Developer-Friendly**: Markdown-based with excellent code highlighting
- **Feature-Rich**: Built-in blog, search, tags, and categories
- **Fast**: Quick builds and instant navigation
- **Flexible**: Easy to customize and extend

## Code Example

Here's a quick Python example showing syntax highlighting:

```python
def greet(name: str) -> str:
    """
    Greet someone by name.
    
    Args:
        name: The person's name
        
    Returns:
        A greeting message
    """
    return f"Hello, {name}! Welcome to the blog."

if __name__ == "__main__":
    message = greet("World")
    print(message)
```
